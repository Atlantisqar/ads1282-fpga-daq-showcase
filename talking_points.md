# 面试讲解稿（公开展示版）

## 1 分钟项目概述

我做了一个基于 FPGA 的三通道高精度数据采集系统。系统用 FPGA 控制三路 ADC 同步采样，把数据组织成固定长度数据块，同时走两条路径：一条通过局域网 UDP 实时发送到 PC 软件显示波形和噪声指标，另一条写入 SD 卡做本地记录。PC 软件支持设备状态、参数配置、实时波形、真实 RMS、等效 RMS 噪声、SNR 和数据保存。

这个项目对我最大的锻炼是完整系统联调：既要理解板级硬件接口，也要写 FPGA RTL，还要做网络、SD、桌面软件和上板调试。过程中遇到过网络不通、SD 写入失败、ADC 同步异常、LED 状态误导、资源接近上限等问题，我都通过状态上报、示波器、抓包和软件对照逐步定位。

## 3 分钟技术讲法

这个项目我按“采集、缓存、传输、记录、显示、诊断”来拆。

采集侧，FPGA 负责三路 ADC 的同步控制和数据读取。这里重点不是单路能不能读到数据，而是三路数据边界是否一致、配置是否回读匹配、异常是否能被上报。

数据组织侧，FPGA 把采样点组织成固定长度二进制数据块。这样网络发送和 SD 写入可以共用同一份数据流，PC 端也容易解析和保存。

网络侧，采用 UDP 做实时传输。UDP 的优势是延迟低、资源可控，适合采集类流式数据。可靠性通过序号、状态、上位机统计和缓存溢出标志来观察。

SD 侧，采用 SPI 模式顺序写入。它和网络链路并行，不反压核心采集链路。这样即使 SD 卡写入慢或异常，实时网络采集仍能继续，异常通过状态和计数暴露出来。

软件侧，我做了 Qt 桌面工具，可以看设备状态、改采样参数、看实时波形、统计真实 RMS、等效 RMS 噪声、峰峰值和 SNR，也能保存数据和处理 SD 记录。

调试上，我尽量把 FPGA 内部不可见状态外显出来：网络状态、ADC 配置、SD 状态、溢出、同步异常都能在软件或 LED 中看到。这样问题定位不是靠猜，而是按证据推进。

## FPGA 岗位深挖讲法

如果面试官继续问 FPGA 细节，我会重点讲四点：

1. 数据链路分层：ADC 控制、数据块组织、缓存、UDP、SD 和状态诊断互相解耦。
2. CDC：网络 PHY 时钟域和应用采集时钟域隔离，多 bit 数据走 FIFO，单 bit 事件走同步器。
3. 配置闭环：软件发命令后，FPGA 接受、触发 ADC 重新配置、回读确认、丢弃稳定样本，再恢复输出。
4. 资源优化：BRAM 是主要瓶颈，所以缩小大 FIFO、把小缓存改为分布式 RAM，保留必要缓冲，避免为了调试把资源堆满。

公开讲解中不展开完整状态机、完整命令编号和完整数据格式，只讲工程设计方法。

## 单板硬件岗位深挖讲法

如果面试官偏硬件，我会重点讲：

1. 时钟复位：系统不只一个时钟域，复位要分域同步，外设释放顺序要考虑。
2. ADC 接口：关注采样时钟、同步信号、数据就绪和回读，不只看是否有数据。
3. 以太网 PHY：按电源、复位、参考时钟、RMII 活动、ARP、UDP 分层排查。
4. SD 卡：初始化成功不等于写入成功，要看写命令响应、忙状态释放和地址模式。
5. 仪器闭环：示波器、抓包、PC 软件状态和 FPGA 内部状态要能互相印证。

## 常见问题回答

### 这个项目最有价值的地方是什么？

不是单个模块，而是完整闭环：从 ADC 到 FPGA、从 FPGA 到网络和 SD、从 PC 软件到数据分析，再到示波器和抓包验证。它覆盖了硬件接口、RTL、协议、软件和调试方法。

### 为什么用 UDP？

采集数据更关注持续流和低延迟。UDP 资源开销小，FPGA 实现可控。丢包或缓存异常通过序号和状态统计观察，而不是依赖复杂重传机制。

### 为什么 SD 不做普通文件？

第一阶段目标是验证连续记录链路，因此采用更直接的顺序写入方式。这样 FPGA 侧逻辑简单、资源可控，PC 端通过工具提取。后续如果产品化，可以再考虑文件系统层。

### 怎么证明数据不是假的？

通过多层证据：ADC 配置回读、采样计数、数据块序号、PC 波形、噪声统计、SD 提取结果、示波器信号和网络抓包。不是只看一个软件窗口。

### 遇到资源不够怎么办？

先看瓶颈是哪类资源。这个项目里片上存储比 LUT 更紧张，所以优先优化 FIFO 深度和缓存实现方式，而不是盲目裁剪功能。

### 如果公开项目，哪些不能讲？

不能公开完整协议、完整寄存器序列、完整状态机、完整引脚表、私有地址、网表和可直接复刻的参数。可以讲系统目标、分层架构、调试方法、指标和个人贡献。

## 结束语

这个项目体现的是我从硬件接口到 FPGA RTL、从上板调试到 PC 软件工具的系统工程能力。它不是只完成一个 demo，而是把问题定位、状态可视化、数据保存和资源优化都纳入了设计。

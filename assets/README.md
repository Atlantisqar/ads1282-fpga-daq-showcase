# 展示图片说明

本目录存放 GitHub 公开展示版使用的图片。当前图片已按公开展示需求处理，可以直接在文档和网页中引用。

## 公开展示注意事项

- 图片中不应出现完整工程路径、私有 IP/MAC、完整协议 payload、完整引脚表或网表内容。
- 若后续替换图片，请继续遮挡芯片敏感丝印、二维码、内部版本号和不可公开的仪器窗口信息。
- 文件名保持稳定，便于 `README.md`、`project_showcase.html` 和其他 Markdown 文档持续引用。

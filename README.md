# 三通道高精度 FPGA 数据采集系统

这是一个面向高精度传感测量场景的三通道数据采集项目。系统以 FPGA 为实时核心，完成三路 ADC 同步采集、实时数据组织、局域网 UDP 传输、SD 卡本地记录，以及 PC 桌面软件的波形显示、状态监控、噪声统计和数据保存。

本仓库是公开项目展示版，只展示系统能力、工程方法、调试过程和验证结果，不公开完整源码、完整协议、精确引脚、私有网络参数、网表和可直接复刻的底层实现细节。

## 项目能力摘要

- 三通道同步 ADC 采集，支持参数配置、回读校验和异常状态上报。
- FPGA 内部完成固定长度数据块组织，并行输出到实时网络链路和本地 SD 记录链路。
- UDP 实时传输用于 PC 端在线观察，SD 顺序记录用于离线数据保存。
- Qt 桌面软件支持设备状态、采样参数、实时波形、真实 RMS、等效 RMS 噪声、峰峰值、SNR、CSV/RAW 保存和 SD 数据提取。
- 上板调试覆盖时钟复位、ADC 数字接口、以太网 PHY、SD 卡、LED 状态和 PC 软件闭环。

## 系统数据链路

```text
三路 ADC
  -> FPGA 同步采集控制
  -> 固定长度数据块组织
     -> UDP 实时传输 -> PC 软件显示与保存
     -> SD 本地记录 -> 离线数据提取
```
